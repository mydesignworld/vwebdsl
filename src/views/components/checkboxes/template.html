Switches & Checkboxes allow users to select one or more items from a set. It can also be used to turn an option on or off.

Switches & Checkboxes have been in user interfaces for a long time and should be used as expected.

It should be easily visible if a switch or checkbox has been selected, and selected items should be more visually prominent than unselected items.

Switches & Checkboxes should make it easy to compare and obvious as to what the available options are.

Name Type Description Accepted
switch Boolean Will set the checkboxes to a switch L&F state false, true
button Boolean Will set checkbox to a selectable button L&F state false, true
inline Boolean Will set the checkboxes to a inline (horizontal) L&F state false, true
single Boolean Set the checkbox to a select one option state, where selecting one option will deselect other options. Similar to radion button functionality false, true
