Tables present a list of organised information in a grid format, and consist of rows and columns. It always have a header and can incude a footer to display totals. The goal is to organise data in a logical way that makes it easy for the user to scan and look for patterns. In addition to data it can contain actionable components like buttons and dropdowns.

Data should be sorted in a logical way that can be by date, alphabetically or numerically.

The user should be able to interact wit the information, e.g. being able to sort it or to trigger and action.

Data should be structured in a logical way to allow the user to easily understand the it.
Name Type Description Accepted
:headers Array Indictate the array in the data file that should be used to create the individual headers
:rows Array Indictate the array in the data file that should be used to create the table's rows/columns and content
:footers Array Indictate the array in the data file that should be used to create the table's footer columns and content
:sort Object Indicate if the table should be sortable and what data objects should be used. The data object will define what field should be the default sort and if it should be ascending (asc) or descending (desc) :sort="custom",
custom:{field: 'amt', order: 'asc'}
:pagination Object Indicate if the table should be able to paginate and what data objects should be used. The data object will determine if pagination is enabled (enabled: true), how many items per page (itemsPerPage: 5), and the position of the pagination bar (align: 'right') :pagination="custom",
custom:{enabled: true, itemsPerPage: 5, align: 'right'}
responsive Boolean Will make the table responsive and add horizontal scroll, should the table exceed the page/container width false, true
scrollable Boolean Will limit the table to 10 rows, and enable scrolling the view the rest of the data false, true
hover Boolean This will add a roll-over animation on each row to increase row readability false, true
tableStyle String This is a placeholder, should the need arise to add any additional styling to set table Any String
