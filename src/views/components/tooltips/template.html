Tooltips are used to display informative text or html when users hover over it, or tap to an element.

Tooltips appear on hover, focus, or touch, and disappear when cursor moves away from HTML element.

A tooltips must be in close proximity to the element with which they are associated.

Tooltips can include text or HTML and must be descriptive in relation to the content to which it's attached too.

Name Type Description Accepted

position String Will determine the position of the tooltip top, right, bottom, left
